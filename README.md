# 📚 Pubs Sales Dashboard

A Business Intelligence project built using **Power BI**, **SQL Server**, and **DAX** based on the Microsoft **Pubs** sample database.

## 📖 Project Overview

This project analyzes book sales data from the Microsoft Pubs database and transforms raw transactional data into an interactive Power BI dashboard.

The dashboard helps users explore sales performance, book categories, publishers, stores, authors, and sales trends through dynamic visualizations and KPIs.
